# Gopter

## Resume
A simple POC of scantime crypter in Golang in just 110 LOC (comments included)

## Features
- [x] Scantime
- [x] \*Nix support
- [x] Drop and launch

## Usage
```
$ git clone https://github.com/Ogromny/Gopter
$ cd Gopter
$ ./build.sh
$ ./builder < yourFile (you can test with sample.elf generated with ./build_sample_file.sh)
$ ./encrypted.elf
```
